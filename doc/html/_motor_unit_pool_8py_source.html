18 #
19 # \f{equation}{
20 # F_{sat} = \frac{1-e^{-b.force}}{1+e^{-b.force}}
21 # \f}
22 #
23 # - Inputs:
24 # + **force**: force before the saturation.

26 # + **b**: saturation function parameter.
27 #
28 # - Outputs:
29 # + Saturated force.
30 #

32  return 2.0 / (1 + np.exp(-b * force)) - 1
