13 # Computes the leak conductance of the compartment.
14 #
15 # - Input:
16 # + **area**: area of the compartment in cm\f$^2\f$.
17 # + **specificRes**: specific resistance of the compartment
18 # in \f$\Omega.cm^2\f$.
19 #
20 # - Output:
21 # + Leak conductance in MS.
22 #
23 # It is compute according to the following formula:
24 #
25 # \f{equation}{
26 # g = 10^6 . \frac{A}{\rho}
27 # \f}
28 # where \f$A\f$ is the compartment area [cm\f$^2\f$], \f$\rho\f$ is
29 # the specific resistance [\f$\Omega.cm^2\f$] and \f$g\f$ is the
30 # compartment conductance [MS].
31 #
32 def calcGLeak(area, specificRes):
33  return (1e6 * area) / specificRes
