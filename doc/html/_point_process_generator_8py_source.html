17 # Generates a number according to a Gamma Distribution with an integer order **GammaOrder**.
18 #
19 # - Inputs:
20 # + **GammaOrder**: integer order of the Gamma distribution.
21 #
22 # + **GammaOrderInv**: inverse of the GammaOrder. This is necessary
23 # for computational efficiency.
24 #
25 # - Outputs:
26 # + The number generated from the Gamma distribution.
27 #
28 # The number is generated according to:
29 #
30 # \f{equation}{
31 # \Gamma = -\frac{1}{\lambda}\ln(\limits\prod_{i=1}^{\lambda} U(0,1))
32 # \f}
33 # where \f$\lambda\f$ is the order of the Gamma distribution and U(a,b) is
34 # a uniform distribution from a to b.
35 #
36 #
37 def gammaPoint(GammaOrder, GammaOrderInv):
38  aux = np.prod(np.random.uniform(0.0, 1.0, GammaOrder))
39 
40  return - GammaOrderInv * math.log(aux)
