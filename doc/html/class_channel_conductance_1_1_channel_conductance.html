Builds an ionic channel conductance.

Inputs: kind - string with the type of the ionic channel (Na, Ks, Kf or Ca) conf - instance of the Configuration class (see Configuration file) compArea - float with the area of the compartment that the Channel belongs, in cm2 pool - the pool that this state belongs. index - the index of the unit that this state belongs.
